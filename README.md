# 4. Miniguia de Estudo

## Resumo Estruturado

### O que é Pixel Art?

Pixel Art é uma técnica de arte digital em que as imagens são construídas manualmente utilizando pixels individuais como unidades básicas de composição.

### Principais Características

* Baixa resolução.
* Controle manual dos pixels.
* Estilo visual retrô.
* Uso de paletas limitadas de cores.
* Grande atenção aos detalhes.

### Processo Básico de Criação

#### 1. Definir o tamanho do projeto

Exemplos:

* 16x16 pixels
* 32x32 pixels
* 64x64 pixels

#### 2. Criar a silhueta

Construir a forma principal do objeto ou personagem.

#### 3. Adicionar contornos

Definir melhor os limites visuais.

#### 4. Aplicar cores base

Utilizar uma paleta reduzida e consistente.

#### 5. Inserir sombras e luzes

Criar profundidade e volume.

#### 6. Refinar detalhes

Eliminar pixels desnecessários e melhorar a legibilidade.

---

## Boas Práticas

* Trabalhar inicialmente com poucas cores.
* Priorizar formas simples.
* Utilizar referências visuais.
* Manter consistência entre sprites.
* Ampliar a imagem apenas para edição, sem alterar a resolução original.

---

## Erros Comuns

* Excesso de detalhes em resoluções pequenas.
* Uso exagerado de cores.
* Sombras inconsistentes.
* Contornos irregulares.
* Escala inconsistente entre elementos.

# Glossário

| Conceito        | Definição                                                     |
| --------------- | ------------------------------------------------------------- |
| Pixel           | Menor unidade de uma imagem digital.                          |
| Sprite          | Personagem ou objeto utilizado em jogos digitais.             |
| Paleta de Cores | Conjunto de cores utilizadas na arte.                         |
| Anti-Aliasing   | Técnica para suavizar bordas.                                 |
| Dithering       | Técnica que mistura cores usando padrões de pixels.           |
| Outline         | Contorno externo de um objeto.                                |
| Resolução       | Quantidade de pixels que compõem a imagem.                    |
| Tileset         | Conjunto de blocos reutilizáveis para construção de cenários. |
| Sombreamento    | Aplicação de sombras para criar profundidade.                 |
| Iluminação      | Definição da origem da luz na composição.                     |
